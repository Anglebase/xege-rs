xege::mat::dynamic

Struct Mat

pub struct Mat<T> { /* private fields */ }
Expand description

A dynamic size matrix of row primary order for an M x N.

It has the same memory layout as Mat<T, M, N>, and can also interact with the API by taking the address of the element Mat[(0,0)].

§Generic Parameters

  • T : The type of matrix element
